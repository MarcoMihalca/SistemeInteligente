# SistemeInteligente: AI Based Burnout Prediction
**Starea de bine, un lux in era Digitala:** Predictii pentru Riscul de Burnout, prin intermediul Indicatorilor de Sanatate.

**Burnout-ul**: o perioada neplacut, nu neasteptata si matematic inevitabila.

## Obiectivul Proiectului

Acest proiect doreste sa prezica cat de predispus este un angajat, din domeniul remote, la "work burnout". Totul depinde de factori specifici, precum pozitia de lucru a acestuia, experienta si multi alti factori specifici fiecarui angajat, precum posibile probleme legate de sanatatea mintala, factori care pot sa accelereze acest fenomen. Nu doar orele peste program cauzeaza burnout, ci chiar si degradarea sănătății fizice (lipsa miscarii, ergonomia proasta) accelereaza epuizarea mentala cu un anumit procentaj. Proiectul va folosi algoritmi pentru a putea imbina datele de sanatate fizica, cu metricile de performanta profesionala, acesta identificand tipare subtile care preced instalarea epuizarii cronice. Scopul final nu este doar monitorizarea, ci crearea unui sistem digital de lucru care prioritizeaza sanatatea umana, transformand 'starea de bine' dintr-un lux intr-un standard operational, deoarece burnout-ul nu ar trebui gandit ca un moment care trebuie sa fie rezolvat/tratat de catre angajator cand acesta este prezent, el trebuie sa fie prevenit prin interventii timpurii si posibile recomandari de solutii care pot fi identificate automat de catre sistem, diferit pentru fiecare angajat, in functie de profilul fiecaruia.
